Power BI

This is as demo for embedding Power BI reports into a .NET MVC Web App through an iframe

Instructions

  • Power BI Pro is required, or at least a trail. Report needs to be published to Power BI online.
  • Then from Power BI online go to the File menu and choose 'Embed'. This will generate a url and a html iframe tag
  • The url can just be pasted in any browser and the report will be generated
  • The tag is used to embed the actual Power BI into your web application
